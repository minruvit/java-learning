# 변수는 선언될 때의 Type에 따라 저장할 수 있는 값의 종류, 허용 범위가 달라진다.
## Primitive Type Variable(기본 타입 변수)
- 8개의 기본 타입

| 값의 분류          | 기본 타입                        |
|----------------|------------------------------|
| 정수             | byte, char, short, int, long |
| 실수             | float, double                |
| 논리(true/false) | boolean                      |
